# Language Preferences

## Communication Language
- Always communicate in **English** for chat interactions
- Use English for code comments and technical discussions
- Keep all code identifiers (variables, functions, classes) in English

## Documentation Language
- Generate spec definition documents in **Spanish**
- Write requirements and design documents in **Spanish**
- User-facing documentation should be in **Spanish**

## Code vs Documentation
- Code: English (including comments, variable names, function names)
- Specs and requirements: Spanish
- Chat responses: English
